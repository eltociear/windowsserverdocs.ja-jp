-   AD DS の展開の詳細については、「Windows Server 2008 Active Directory Domain Services のインストールと削除」 ([https://go.microsoft.com/fwlink/?LinkId=100492](https://go.microsoft.com/fwlink/?LinkId=100492)) の手順ガイドを参照してください。
