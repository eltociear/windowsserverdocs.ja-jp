# <a name="bitsadmin-cache-and-help"></a>bitsadmin cache とヘルプ

**キャッシュ**スイッチのコマンドラインの使用法を表示します。

## <a name="syntax"></a>構文

```
bitsadmin /cache /help
```

## <a name="examples"></a><a name=BKMK_examples></a>例

次のコマンドは、**キャッシュ**スイッチのコマンドラインヘルプを表示します。

```
bitsadmin /cache /help
```
