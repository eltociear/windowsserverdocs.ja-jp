# <a name="bitsadmin-util-and-help"></a>bitsadmin util および help

**/Util**スイッチのコマンドラインの使用法を表示します。

## <a name="syntax"></a>構文

```
bitsadmin /util /help
```

## <a name="examples"></a>例

**/Util**スイッチのコマンドラインヘルプを表示するには、次のようにします。

```
bitsadmin /util /help
```
