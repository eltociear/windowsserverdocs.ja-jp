# <a name="nslookup-set-vc"></a>nslookup set vc



サーバーに要求を送信するときに、仮想回線を使用するかどうかを指定します。

## <a name="syntax"></a>構文

```
set [no]vc
```

### <a name="parameters"></a>パラメーター

| パラメーター |                                              説明                                               |
|-----------|--------------------------------------------------------------------------------------------------------|
| **novc**  | サーバーに要求を送信するときに、仮想回線を使用しないように指定します。 既定値は**novc**です。 |
|  **vc-1**   |             サーバーに要求を送信するときに常に仮想回線を使用するように指定します。             |
|   {ヘルプ   |                                                   ?}                                                   |
