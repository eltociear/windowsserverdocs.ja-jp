# <a name="nslookup-set-all"></a>nslookup set all



構成設定の現在の値を出力します。

## <a name="syntax"></a>構文

```
set all 
```

### <a name="parameters"></a>パラメーター

| パラメーター | [説明] |
|-----------|-------------|
|   {ヘルプ   |     ?}      |

## <a name="remarks"></a>Remarks

-   [**すべて設定**] では、既定のサーバーとコンピューター (つまり、ホスト) に関する情報も印刷されます。
