# <a name="nslookup-set-port"></a>nslookup set port



既定の TCP/UDP ドメインネームシステム (DNS) ネームサーバーポートを、指定された値に変更します。

## <a name="syntax"></a>構文

```
set port=<Port>
```

### <a name="parameters"></a>パラメーター

| パラメーター |                                          [説明]                                          |
|-----------|-----------------------------------------------------------------------------------------------|
|  \<ポート>  | 既定の TCP/UDP DNS ネームサーバーポートの新しい値を指定します。 既定のポートは53です。 |
|   {ヘルプ   |                                              ?}                                               |
