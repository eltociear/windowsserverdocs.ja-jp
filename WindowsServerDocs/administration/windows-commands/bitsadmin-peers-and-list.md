# <a name="bitsadmin-peers-and-list"></a>bitsadmin のピアとリスト

すべてのピアを一覧表示します。

## <a name="syntax"></a>構文

```
bitsadmin /peers /list
```

## <a name="examples"></a><a name=BKMK_examples></a>例

次の例では、すべてのピアを一覧表示します。

```
C:\>bitsadmin /peers /list
```
