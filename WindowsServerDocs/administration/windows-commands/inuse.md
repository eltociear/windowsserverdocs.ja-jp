# <a name="inuse"></a>inuse



Inuse は非推奨とされており、Windows の将来のリリースでサポートされるとは限りません。

このツールは Windows Server 2003 に含まれています。 詳細については、「 [Inuse](https://technet.microsoft.com/library/dd996699(v=ws.10).aspx)」を参照してください。
