# <a name="online"></a>online



オフラインのディスクまたはボリュームをオンライン状態にかかります。

## <a name="syntax"></a>構文

```
Offline disk
Offline volume
```

### <a name="parameters"></a>パラメーター

|パラメーター|説明|
|---------|-----------|
|[オフラインディスク](offline-disk.md)|オンライン状態にフォーカスがあるオフライン ディスクを移動します。|
|[オフラインボリューム](offline-volume.md)|オンライン状態に、オフラインのボリュームがフォーカスを移動します。|
