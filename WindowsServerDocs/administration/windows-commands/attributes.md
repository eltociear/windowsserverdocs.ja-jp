# <a name="attributes"></a>属性

ディスクまたはボリュームの属性を表示、設定、またはクリアします。

## <a name="syntax"></a>構文

```
Attributes disk
Attributes volume
```

### <a name="parameters"></a>パラメーター

| パラメーター | 説明 |
| --------- | ----------- |
| [属性ディスク](attributes-disk.md) | ディスクの属性を表示、設定、またはクリアします。 |
| [属性のボリューム](attributes-volume.md) | ボリュームの属性を表示、設定、またはクリアします。 |
