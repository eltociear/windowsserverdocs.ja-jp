# <a name="bitsadmin-peers-and-discover"></a>bitsadmin ピアと検出

ピアを再び検出します。

## <a name="syntax"></a>構文

```
bitsadmin /peers /discover
```

## <a name="examples"></a><a name=BKMK_examples></a>例

次の例では、ピアを検出します。

```
C:\>bitsadmin /peers /discover
```
