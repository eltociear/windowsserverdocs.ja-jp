# <a name="begin-restore"></a>復元の開始

> 適用対象: Windows Server 2012 R2、Windows Server 2012

復元セッションを開始し、関連するライターに**復元前**イベントを発行します。

## <a name="syntax"></a>構文

```
begin restore
```
