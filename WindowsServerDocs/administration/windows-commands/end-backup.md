# <a name="end-backup"></a>バックアップの終了




完全バックアップセッションを終了し、必要に応じて、適切なライター状態の**BackupComplete**イベントを発行します。

## <a name="syntax"></a>構文

```
end backup
```

## <a name="remarks"></a>コメント

-   このコマンドは、すべての完全バックアップに使用する必要があります。
