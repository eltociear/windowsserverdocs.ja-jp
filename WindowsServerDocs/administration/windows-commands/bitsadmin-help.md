# <a name="bitsadmin-help"></a>bitsadmin help

Bitsadmin のコマンドラインパラメーターおよびオプションに関するヘルプ関連情報を表示します。

## <a name="syntax"></a>構文

```
bitsadmin /help | /?
```

## <a name="examples"></a><a name=BKMK_examples></a>例

次の例では、コマンドラインヘルプを取得します。

```
C:\>bitsadmin /help
```
