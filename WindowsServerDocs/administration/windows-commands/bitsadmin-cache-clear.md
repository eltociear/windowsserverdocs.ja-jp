# <a name="bitsadmin-cache-and-clear"></a>bitsadmin cache と clear

ローカルキャッシュを削除します。

## <a name="syntax"></a>構文

```
bitsadmin /cache /clear
```

## <a name="examples"></a><a name=BKMK_examples></a>例

次の例では、ローカルキャッシュを削除します。

```
C:\>bitsadmin /cache /clear
```
